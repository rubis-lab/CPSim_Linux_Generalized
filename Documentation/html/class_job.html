Detailed Description

Header file for Engine-Job A job is the instance of certain task. So it has dynamical properties with the task. Job can have below properties.

  1. Name, of the task which is instantiated.
  2. Release time,
  3. Deadline,
  4. Start time ranges[EST, LST]
  5. Finish time ranges[EFT, LFT]
  6. Priority
